Users Orders Offers

Available API methods

/users - GET, POST
/users/id - GET, PUT, DELETE
/orders - GET, POST
/orders/id - GET, PUT, DELETE
/offers - GET, POST
/offers/id - GET, PUT, DELETE

When adding orders with "/orders POST method" or updating orders with "/orders/id PUT method" make sure dates are submitted as MM/DD/YYYY
